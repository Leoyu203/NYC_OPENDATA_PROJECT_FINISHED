This dataset contains all job applications submitted through the Borough Offices, through eFiling, or through the HUB, which have a "Latest Action Date" since January 1, 2000. It doesn't include Post Approval Amendments (PAAs). However, it does include detailed information on an applicant's status, location, job type, and filing method. This dataset is useful to hold records and allow applicants to submit job applications. For instance, reviewing job filing patterns across the city and its time periods. Each record in the dataset includes property address, owner and applicant details, and the application status. The dataset is updated regularly and it is a critical resource for those in agriculture, tech, engerinning, resources, as well as, the general public. In this info page, you can filter information based either the Borough or the job type, and then you can see where they applied this job application using the map given on the right.
